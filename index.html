    Project Overview

    Empowering Everyone
    With Accessible AI

    AccessAI NextGen is built with a singular vision: to make the web accessible to everyone, regardless of physical or sensory limitations.

    By combining high-speed browser-based AI systems (MediaPipe & TensorFlow.js) with standard speech interface components, we provide an interactive hands-free web environment that removes traditional accessibility boundaries.

    Technical Stack Architecture

    1

    MediaPipe Hands

    Processes standard camera input frames instantly to locate 21 key points, enabling precision-grade hand skeleton modeling without latency.

    2

    TensorFlow.js & Custom Heuristics

    Computes comparative angles, Euclidean joint separations, and vector distributions inside an offline classification pipeline for error-free sign decoding.

    3

    Speech Recognition & Synthesis

    Operates continuously in the background to handle user routing instructions and screen reading announcements seamlessly.
